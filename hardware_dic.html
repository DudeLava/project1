The Basic Terms

Learn Some Basic Termenology

CPU (Central Processing Unit)
The CPU is the brain of the computer responsible for executing instructions and performing calculations.
RAM (Random Access Memory)
RAM is the temporary storage space used by the computer to store data and instructions that are actively being used.
Hard Drive
The hard drive is a non-volatile storage device used to store data and programs on a long-term basis.
Motherboard
The motherboard is the main circuit board that connects and allows communication between various hardware components of a computer.
GPU (Graphics Processing Unit)
The GPU is a specialized processor that handles complex graphical computations and is primarily used for rendering images and videos.
Power Supply Unit (PSU)
PSU is responsible for converting AC power from an outlet into DC power for powering computer.
