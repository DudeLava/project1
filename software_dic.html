The Basic Terms

Learn Some Basic Termenology

Operating System
Software that manages computer hardware and provides a user interface.
Application
Software designed to perform specific tasks or functions for users.
Compiler
Software that converts human-readable code into machine-readable code.
Debugger
Software tool used to identify and fix errors in programs.
Database
Software that stores, organizes, and retrieves large amounts of data.
Web Browser
Type Of Software that is used to access and navigate through the internet.
